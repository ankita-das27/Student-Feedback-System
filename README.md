# Student Feedback System (Django)

## Description
A simple Django web application where users can submit feedback. The feedback is stored in a database and can be viewed by the admin through the Django admin panel.

## Features
- Feedback submission form
- Database storage using SQLite
- Admin panel for managing feedback

## Technologies Used
- Python (Django)
- HTML
- SQLite

## How to Run
```bash
pip install django
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
